Class: List

conbo.List(options)

new List(options)

A bindable Array wrapper that can be used when you don't require web service connectivity. Plain objects will automatically be converted into an instance of the specified `itemClass` when added to a List, and the appropriate events dispatched if the items it contains are changed or updated.
Parameters:
Name Type Description
options Object Object containing optional initialisation options, including `source` (array), `context` (Context) and `itemClass` (Class)
