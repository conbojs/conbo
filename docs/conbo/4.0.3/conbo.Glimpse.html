Class: Glimpse

conbo.Glimpse

new Glimpse(options)

Glimpse A lightweight element wrapper that has no dependencies, no context and no data binding, but is able to apply a super-simple template. It's invisible to View, so it's great for creating components, and you can bind data to it using the `cb-data` attribute to set the data property of your Glimpse
