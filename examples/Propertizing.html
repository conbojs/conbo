Propertize

Hello, my name is and this is an example of how to use my amazing ability to propertize your classes!

Propertizing is when Conbo automatically converts the get_propName and set_propName methods in your class into properties, e.g. propName. If your class extends EventDispatcher, these properties will automatically be bindable, so that changes can be detected without using this.set(...);.

This example app has a read-only method named get_numChildren, which counts the number of DOM elements it contains: app.numChildren =

You can propertize non-Conbo objects using conbo.propertize(obj);
