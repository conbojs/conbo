Propertize

Hello, my name is and this is an example of how to use my amazing ability to propertize your classes!

Propertizing is when Conbo automatically converts the get_propName and set_propName methods in your class into properties, e.g. propName. If your class extends Bindable, these properties will automatically be bindable, so that changes can automatically be detected even when you're not using this.set().
